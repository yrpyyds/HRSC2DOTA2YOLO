# HRSC2016数据集以及DOTA数据集转为YOLOv8旋转框标注格式
配备了转换脚本以及显示标注图像的UI(基于pyqt),代码很简单,有较详细的中文注释,直接使用即可
注意你所需要的类别,本脚本将HRSC2016的三级类别归到了二级类别,且将一级类别ship删去,可自行修改需要保留的类别
